# SRCINFO
作为白帽子，手上有很多应急响应中心接受漏洞的范围和一些网站信息，为了便于管理，自己动手写了一个信息管理的平台，感兴趣的可以自己改改，制作自己的信息库
以下为一些截图。毕竟这种东西比的就是谁的自动化好，自动化收集信息脚本就不给大家了。
